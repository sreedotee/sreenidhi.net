Tofha Web Design

Tofha is a handmade gifting brand by an artist. Most of its customers were already finding products through Instagram, DMing the artist with an occasion or a person in mind, talking through options, then checking out. We designed and built the website as an extension of that flow, preserving the conversational, exploratory shopping mode while turning it into a self-serve experience.

The homepage is built around how Tofha customers actually think. Items are grouped by occasion, by recipient, and by region, with bestsellers surfaced on the landing surface itself. Each grouping targets a different audience mental model, so visitors land somewhere relevant on first scroll instead of being dropped into a flat product grid.

Work in progress

Shoppers arrive in one of two modes. Some know exactly what they want, like a specific flower or product. Others are browsing for a person or an occasion and don't yet know what they're looking for. The shop UI separates these into two siloed modes that the visitor toggles between, each with its own filter set and entry points, so the interaction model matches the intent on arrival instead of forcing both audiences through the same flow.

Tofha also runs in-person workshops led by the artist. The site includes an end-to-end booking flow, so workshop discovery, scheduling, and checkout sit inside the same experience as product purchase.

We handled both the design and the development, built on Shopify. The catalog setup involved data modeling around products and variants, deciding what counts as one product with options versus distinct products, so the storefront stayed browsable without inflating SKU counts. Inventory upload and store configuration were automated via the Shopify Admin API and Shopify's dev MCP tooling, which let us script bulk operations and run programmatic checks during setup.
